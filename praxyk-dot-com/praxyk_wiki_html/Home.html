Overview

Downloading the Project

git clone https://github.com/Praxyk/Praxyk.git

Build and Install

cd praxyk
./build.sh

Highest Level Directories

  • POD - Highest directory for all pre-trained ML-container projects.
  • TLP - High directory for all template ML-container projects.
  • API - Contains all code for the API server.
  • models - Contains all Models for the project.
  • queue - Contains the code for the task-queue.
