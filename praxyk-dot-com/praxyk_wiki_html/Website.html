###Website Server Setup# The website is up as of Oct 12th. May be down intermittently. Home Page

####Requirements On a fresh Ubuntu 14.04 x64 instance, use the following install script to setup the server environment.

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install lighttpd

####Setup

rm /var/www/index.lighttpd.html
git clone -b website http://www.github.com/praxyk.git
sudo rsync -rtv /home/{user}/praxyk/website/ /var/www/

In order to auto update the website every hour do the following:

sudo crontab -e

A dialog will ask what editor to use. Then go down to the bottom of the file and add the following:

0 * * * * git -C /home/{user}/praxyk pull
0 * * * * rsync -rtv /home/{user}/praxyk/website/ /var/www/

The server will now be set up.
