Optical Chacter Recognition

Optical Character Recognition is a Machine Learning model that can detect and extract text that is present in a given image.

It works as follow :

  • Upload of Set of Images to Perform OCR on.
  • We run each of the images through our OCR model and extract the text present.
  • We return to you a string for each image containing our best guess as to the text present in the images you submitted.
