Environment and Build Config

This page describes how to configure the various environments necessary to run the separate parts of this project.

General
  • Distro - Ubuntu 14.04 x64 LTS
  • Python - Python 2.7
Build From Scratch

The entire project is set up so it can be built from scratch on a clean Ubuntu instance that has only git installed. Simply do the following -

sudo apt-get update
sudo apt-get install git

git clone https://github.com/praxyk/praxyk.git
cd praxyk
./build.sh # will take 5-10 minutes to build

After build.sh finished, it will return either success or failure. If it succeeded, you can go ahead and run the API server, the task-queues, the Machine-Learning tasks, etc.

Each top-level directory also has it's own build.sh script that can be called to only build the requirements for that specific directory. More info on these scripts is available in the next section.
